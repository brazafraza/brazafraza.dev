Mind's Isle

Role: Solo project
Engine: Unity


This is a prototype build!

Are you brave enough to uncover the mystery of the mind's isle?
Explore, gather and craft! Explore the island and collect resources, defeat enemies and build structures. Complete quests for the island and make it happy to earn sweet rewards.
The game is not currently still in development as this is only the prototype, however, it will receive updates again in early 2025, revamping the UI and addressing major bugs.

This game tested how I could create functional crafting, inventory, quests, harvesting and grid-building systems. I followed tutorials to create these systems, and by the end having greatly enhanced my c# skills. Another huge accomplishment I made from this project was learning how to use scriptable objects, specifically for my items, crafting recipes and resources, something that will definitely help enhance my future projects.
This project taught me a lot about unity too, using a lot of systems I've never used before such as playing with terrain tool, lighting settings, post-processing, nav mesh and audio systems.
