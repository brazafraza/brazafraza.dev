Beeranged

This is my first-ever devlog post! It will be short as I haven't added too much to the game yet. This post will be more of an introduction and discussing plans about the future of the project.

Beeranged started as an Excel sheet number test project for a shooter game. Its main focus was a test of balancing numbers, seeing how I could make a player and enemies feel balanced while having stats scale over time, without the game feeling stale. I was prototyping the game in an Excel spreadsheet recording player levelling, enemy stat scaling, XP scaling and different player upgrade numbers depending on how many enemies were killed. This felt pretty boring, and as a developer, I thought I might as well create a prototype for the game with a concept I had been thinking about. This led to me prototyping the game in Unity, which turned into something a lot more fun than it sounded on a spreadsheet. It then quickly turned into a test to see how I could make a game with a simple addicting core loop (kill enemy>get Xp> get upgrades), enemy AI, enemy scaling implementation and upgrade systems.

So what's been updated since the release? I've started experimenting with sprite art and these are my first designs for the player, the smoke clouds (bullets) and the enemies. I've done some minor bug fixes, UI polish and changed the dev keys too so if hitting 4 will no longer make you max level, if you hit enough buttons I'm sure you'll figure out how to do it again though :) This marks the first update in what I hope will be a long, rewarding experience as I'm excited to keep adding more in the future. Check out my first attempts at sprite art below!

As for the future of the game, I'd like to create art and polish all aspects of the current game, not changing the actual gameplay too much. Over the next few weeks, I will be working to integrate more audio, and art, potentially adding a level (as in-game level not player) and a leaderboard/score system.

Cheers
- Braza :)
